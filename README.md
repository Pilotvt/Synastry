# Synastry Desktop

Synastry — настольная программа для расчёта личной карты и совместимости пары. Все вычисления офлайн; онлайн нужен только для поиска людей по совместимости и чата (встроенная мини‑соцсеть) и для обновлений. Что делает: строит натальную карту по дате/времени/месту, показывает планеты и дома, считает совместимость (процент + пояснения), отмечает, если данных не хватает. Как считает (по‑человечески): определяет часовой пояс, берёт точные положения планет, привязывает к реальным созвездиям, делит круг на дома — всё локально на вашем ПК. Онлайн‑часть: поиск людей с подходящими картами и переписка прямо в приложении; расчёты работают и без интернета. Системные требования

ОС: Windows 10/11 (x64). Память: от 2 ГБ свободной (приложение и встроенный расчётный движок). Диск: ~1.5–2 ГБ на установку и временные файлы. Процессор: современный x64 с поддержкой SSE2 (обычный офисный/домашний ПК подходит). Интернет: нужен только для регистрации/поиска людей/чата/обновлений; сами расчёты офлайн.
